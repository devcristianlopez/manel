Características de Seguridad Potentes

Todo lo que necesitas para mantener tu entorno de desarrollo seguro.

🔍

Detección de Software

Detecta 21 tecnologías: Node.js, Python, Docker, PostgreSQL, y más — incluyendo gestores de paquetes, bases de datos y herramientas de desarrollo.

🛡️

Análisis de Vulnerabilidades

Consulta las bases de datos OSV, NVD y GHSA para CVEs conocidos en tu stack de software instalado.

🔒

Verificaciones de Hardening

7 verificaciones de seguridad para Linux: estado del firewall, SELinux, configuración SSH, parámetros del kernel y más.

📊

Score de Seguridad

Score ponderado que combina todos los aspectos de seguridad en una métrica (0-100) con recomendaciones accionables.

Inicio Rápido

Comienza en segundos.

terminal 
# Instalar Manel globalmente
npm install -g manel

# Escanear tu sistema
manel scan

# Obtener output en JSON
manel scan --format json

# Verificar estado de hardening
manel hardening

# Ver score de seguridad
manel score

Integración CI/CD

Integra Manel en tus pipelines para verificaciones de seguridad automatizadas en cada commit.

  • Compatible con GitHub Actions
  • Falla builds en vulnerabilidades críticas
  • Sube SARIF a GitHub Code Scanning
  • Funciona con cualquier plataforma CI/CD
.github/workflows/security.yml 
# Ejemplo de GitHub Actions
- name: Security Scan
  run: |
    npm install -g manel
    manel scan --format sarif \
      --output scan.sarif \
      --fail-on critical
