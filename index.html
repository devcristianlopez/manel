Manel — Security Health Monitor

Monitorea la seguridad de tu entorno de desarrollo

Caracteristicas

Escaneo Local

Detecta SO, herramientas de desarrollo, lenguajes y bases de datos instalados via CLI

Vulnerabilidades

Consulta OSV, NVD y GitHub Security Advisories en paralelo

Security Score

Puntuacion 0-100 ponderada con semaforo (verde/amarillo/rojo/negro)

Bases de Datos

PostgreSQL, MySQL, MariaDB, MongoDB, Redis, SQLite, pgAdmin

Hardening

Firewall, SELinux, SSH, puertos, actualizaciones, core dumps

Dashboard

Interfaz visual con Electron + React

Interfaz de Linea de Comandos

Comando Descripcion
manel statusEscaneo rapido del entorno
manel scanEscaneo detallado con vulnerabilidades
manel hardeningChecks de seguridad del sistema
manel runAbre el dashboard Electron
manel helpMuestra ayuda
manel versionVersion instalada

Instalacion

Via npm global (recomendado) 
npm install -g manel
Via script de instalacion 
curl -fsSL https://raw.githubusercontent.com/devcristianlopez/manel/main/setup.sh | bash
Manual 
git clone https://github.com/devcristianlopez/manel.git
cd manel
npm install && npm run build && npm link

Uso rapido

Ver estado de seguridad 
manel status
Escaneo completo 
manel scan
Abrir dashboard 
manel run
